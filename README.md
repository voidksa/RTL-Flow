## ğŸš€ Overview
**RTL Flow** is a powerful browser extension designed to improve the reading experience for Right-to-Left (RTL) language users on the web. It forces correct RTL direction on resistant websites and provides smart formatting for mixed text (English + Arabic/Hebrew/etc.) without breaking the page layout.

## âœ¨ Key Features
- **Smart Mode:** Automatically detects mixed text paragraphs and applies RTL direction only where needed, preserving the original site layout.
- **Force RTL:** Apply RTL direction to any website, even those that explicitly force LTR.
- **Font Improvement:** Enhancing readability with modern fonts like Tajawal, Vazirmatn, and Rubik.
- **Per-Site Settings:** Remembers your preferences for each domain automatically.
- **Keyboard Shortcut:** Quickly toggle RTL on/off:
  - **Windows/Linux:** `Alt+Shift+R`
  - **Mac:** `Option+Shift+R`
